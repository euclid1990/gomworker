# Manager and running multiple Laravel Queue workers

## Install

- [Go](https://golang.org/doc/install)
- [Glide](https://glide.sh)
```terminal
$ glide install
$ glide rebuild
```

## Run

```terminal
$ go run main.go init
$ go run main.go stop
```
